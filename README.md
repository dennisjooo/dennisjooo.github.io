# My Web Portfolio

Well hello there, welcome to the my portofolio's repo. You can check it out [here](https://dennisjooo.github.io).

Beware, I'm not that good at web-design at all. I did this because I was curious of the process in creating a site and all the codes that I need to write.
Previously I've tried making my own profile page as an assignment in CS50, but it was so simple and basically just a compilation of some Bootstrap elements.

I was "inspired" by [this design](https://bootstrapmade.com/devfolio-bootstrap-portfolio-html-template) and there are some elements that was modified from W3Schools for example.
The design as of July 2023 is simple, runs pretty smooth, and mobile-friendly.
